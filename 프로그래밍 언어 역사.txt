1970 C (데니스 리치) - 서로 다른 기종의 컴퓨터에서도 같은 언어로 프로그래밍 가능하도록

1980 C++ (비야네 스트로스트룹) - C언어의 기본을 유지하면서 객체지향의 개념을 도입

1990 Java (제임스 고슬링) - 메모리 관리, 포인터 삭제, 서로 다른 HW/SW에서 똑같이 실행되는 프로그램

JRE(Java Runtime Environment, 자바 실행 환경)
- 자바로 개발된 프로그램을 실행하기 위해 필요한 요소
- 마인크래프트를 실행하기 위해서는 자바가 설치되어 있어야 한다 (이때 자바는 JRE를 말함)
JDK(Java Development Kit, 자바 개발 도구)
- 자바 언어로 프로그램을 개발하기 위한 도구 모음, 실행 및 테스트를 위해 JRE를 포함한다
- 자바 기반의 안드로이드 앱을 개발하기 위해서 자바를 설치해야한다 (이떄 자바는 JDK를 말함)

1990 Python (귀도 반 로썸)
